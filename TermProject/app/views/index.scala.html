Play REST API

This is a placeholder page to show you the REST API. Use httpie to post JSON to the application.

To see all posts, you can do a GET: 

            http GET localhost:9000/posts

To create new posts, do a post 

            http POST localhost:9000/posts title="Some Title" body="Some Body"

You can always look at the API directly: /posts

To see all persons, you can do a GET: 

            http GET localhost:9000/persons

To create new persons, do a post 

            http POST localhost:9000/persons name="Some name"

You can always look at the API directly: /persons
